# Tower Finance - Solidity Contracts

[![Tower Finance](./tower.jpg)](https://towerfinance.io)

## Setup

```
git clone https://github.com/towerfinance/tower-core.git
cd tower-core
npm install
```

## Environment Variables

```
cp .env.example .env
```

## Run test

```
npx hardhat test
```

## Run local

```
npx hardhat node
```
